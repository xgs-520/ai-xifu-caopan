## 📖 简介

「AI媳妇操盘」是一套专业的金融交易方案模板，由 **14个完整章节** 组成，覆盖从行情分析到操作建议的全流程。

**一句话指令，生成专业方案！**

---

## ✨ 核心特色

| 特色 | 说明 |
|-----|------|
| 🎯 **14章节完整覆盖** | 从行情到操作，全流程分析 |
| 📊 **三种操作方案** | 稳健型、激进型、日内短线 |
| 🔗 **产业链分析** | 上游、中游、下游全景 |
| 💰 **席位追踪** | 主力资金动向一目了然 |
| 📱 **手机友好** | 表格化呈现，一目了然 |
| 🤖 **AI智能分析** | 结合AI技术，智能生成方案 |

---

## 📋 方案结构

```
一、行情回顾          → 收盘数据、近期走势
二、股期联动分析      → A股表现、关联品种
三、国际股指期货影响  → 外盘、美股期货
四、宏观影响分析      → 利空/利多因素
五、产业链分析        → 上游、中游、下游
六、市场情绪          → 多空情绪、资金情绪
七、资金动向分析      → 持仓变化
八、席位追踪          → 多空席位、主力动向
九、技术分析          → 关键价位、技术指标
十、操作方案          → 稳健/激进/日内三种
十一、操盘建议        → 资金配置、分步操作
十二、风险提示        → 各类风险及应对
十三、本周关注        → 重要事件
十四、方案总结        → 一句话核心观点
```

## 🔧 配置说明

### 环境变量

| 变量 | 说明 | 获取方式 |
|-----|------|---------|
| `TUSHARE_TOKEN` | Tushare API Token | https://tushare.pro |
| `GS_API_KEY` | 国信证券 API Key | https://www.guosen.com.cn |

### 配置方法

```bash
# 配置 Tushare
export TUSHARE_TOKEN="your_token_here"

# 配置国信证券
export GS_API_KEY="your_api_key_here"
```
