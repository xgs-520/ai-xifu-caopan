# 「AI媳妇操盘」v2.0

## 📋 简介

「AI媳妇操盘」是一套专业的金融交易方案模板，由14个完整章节组成，覆盖从行情分析到操作建议的全流程，帮助用户快速生成专业的期货、股票、基金交易方案。

## ✨ 核心特色

| 特色 | 说明 |
|-----|------|
| **14章节完整覆盖** | 从行情到操作，全流程分析 |
| **三种操作方案** | 稳健型、激进型、日内短线 |
| **产业链分析** | 上游、中游、下游全景 |
| **席位追踪** | 主力资金动向一目了然 |
| **手机友好** | 表格化呈现，一目了然 |
| **AI智能分析** | 结合AI技术，智能生成方案 |

## 📊 方案结构（14章节）

1. **行情回顾** — 收盘数据、近期走势
2. **股期联动分析** — A股表现、关联品种
3. **国际股指期货影响** — 外盘、美股期货
4. **宏观影响分析** — 利空/利多因素
5. **产业链分析** — 上游、中游、下游
6. **市场情绪** — 多空情绪、资金情绪
7. **资金动向分析** — 持仓变化
8. **席位追踪** — 多空席位、主力动向
9. **技术分析** — 关键价位、技术指标
10. **操作方案** — 稳健/激进/日内三种
11. **操盘建议** — 资金配置、分步操作
12. **风险提示** — 各类风险及应对
13. **本周关注** — 重要事件
14. **方案总结** — 一句话核心观点
