## 配置要求

使用前需配置数据接口：

### Tushare（期货数据）

1. 访问 https://tushare.pro 注册
2. 获取 Token
3. 配置环境变量：
   ```bash
   export TUSHARE_TOKEN="your_token"
   ```

### 国信证券（股票行情）

1. 访问 https://www.guosen.com.cn/gs/xxskills/index.html
2. 登录获取 API Key
3. 配置环境变量：
   ```bash
   export GS_API_KEY="your_api_key"
   ```
